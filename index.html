CChip-8

A Chip-8 emulator based on the Telmac 1800. Chip-8 is an interpreted programming language primarily used in the 1980s and 1990s on weaker, DIY home computers. The intent of this program was to expand my knowledge of modern CPU architecture and low-level programming languages. Written in C++.

ErrorHide

A VSCode extension that hides notification popups when a program compiles successfully. Originally intended for LaTeX as minor, immediately fixable errors would cause a compile error which popped up multiple notification boxes. Written in TypeScript.

Linux Quickconfig (LQC)

A Debian and Arch compatible quick config script created in order to quickly install and compile my own dotfiles and forks of st and dwm. Works after an initial base configuration of a Debian or Arch-based distribution. Written in POSIX Shell.
