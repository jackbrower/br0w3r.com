Freedom of Information

A Chromium-based browser addon that disables the paywall before news sites. Works by targeting the elements of the paywall, deleting them, and then re-enabaling the scroll bar. Preferable to disabling JavaScript as this method keeps images and other graphics, where disabling JavaScript can hide these figures. Available on the Chrome Store, or in ZIP format on my GitHub.

visualizer2

A Python program that takes various Calculus equations and visualizes them with MatPlotLib. A user specifies the type of equation they want to visualize (e.g. Reimann sums) and then provides a relevant equation (e.g. x2), along with any other required data. The program then renders a visualization of the equation.
